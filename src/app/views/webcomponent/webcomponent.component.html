Webcomponents

This Webcomponents come from an external github repository, which was added to the package.json file in the angular project folder as dependency. 

"webcomponents": "git@github.com/boendermaker/webcomponents#master"

This needs to be added to the corresponding component modules, to allow custom elements in angular templates. 

import { NgModule, CUSTOM_ELEMENTS_SCHEMA } from '@angular/core';

Webcomponent WebC-ThreeTest ThreeJS Example with Angular Material Sliders

This is a webcomponent with ThreeJS applied to it. The 3D Object will be controlled by angular slider formcontrols, by passing the values over attributes to the webcomponent

Speed of circular movement
Radius of circular movement
Speed of rotation

Webcomponent WebC-Datatest receives data over attributes

The webcomponent takes data over an html attribute and runs its update lifecycle hook after data has changed to run render for diplaying the data as table.
