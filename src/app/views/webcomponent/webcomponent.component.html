Webcomponents

Webcomponent WebC-Datatest receives data over attributes

This Webcomponent comes from an external github repository, which was added to the package.json file in the angular project folder as dependency. 

"webcomponents": "git@github.com/boendermaker/webcomponents#master"

The webcomponent takes data over an html attribute and runs its update lifecycle hook after data has changed to run render for diplaying the data as table. 

import { NgModule, CUSTOM_ELEMENTS_SCHEMA } from '@angular/core';

Needs to be added to the corresponding component modules, to allow custom elements in angular templates.
